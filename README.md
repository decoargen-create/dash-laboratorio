# DASH Laboratorio

Dashboard para gestión de ventas, comisiones y stock de un laboratorio de cosmética.

## Características

- **Panel Admin**: Gestión completa de ventas, productos, clientes, stock y comisiones
- **Panel Mentor**: Vista de solo lectura para que los mentores vean sus comisiones y ventas referidas
- **Cálculo automático de comisiones** al 50% por venta
- **Control de stock** con alertas visuales por nivel bajo
- **CRM de clientes** con asignación a mentores

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Tecnologías

- React 18
- Vite
- Tailwind CSS
- Recharts
- Lucide React Icons
